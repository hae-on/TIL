## 파이썬 기본 자료형

### 자료형 분류

- 불린형
  - True / False
  - 비교/ 논리 연산 수행
  - bool() 함수
- 수치형
  - 모든 정수의 타입은 int
  - 매우 큰 수 오버플로우 발생하지 않음
  - 정수가 아닌 모든 실수는 float
- 문자열
  - 모든 문자는 str 타입
  - 문자열은 작은 따옴표나 큰 따옴표를 활용하여 표기
- None : 값이 없음

### 자료형 변환

- 파이썬에서 데이터 형태는 서로 변환할 수 있음
  - 암시적 형 변환
  - 명시적 형 변환

### 컨테이너

- 시퀀스 (순서 o)
  - 문자열 : 문자들의 나열
  - 리스트 : 변경 가능한 값들의 나열
  - 튜플 : 변경 불가능한 값들의 나열
  - 레인지 : 숫자의 나열
- 컬렉션 (순서 x)
  - 세트(집합) : 유일한 값들의 모음
  - 딕셔너리 : 키-값들의 모음
